### 5. Content Studio (/studio)
**Chat-style interface like Claude.ai**

7-Step Workflow:

**STEP 1: Briefing**
- System suggests 3-5 topics based on user profile
- Asks: objective, available data, preferred topic
- Stores context for next steps

**STEP 2: Structure Selection**
- Offers 3 copywriting structures (PAS, AIDA, BAB, FAB, HSO, Storytelling)
- Indicates recommended option
- User selects one

**STEP 3: Content Type**
- Offers 3 content types from 16 available
- Based on structure + objective
- User selects one

**STEP 4: Hook Creation**
- Generates 3 hooks (authority+data, personal narrative, tension/contrast)
- User can regenerate (max 10x per step)
- Regeneration modes: new, variations, specific feedback

**STEP 5: Body Construction**
- Builds complete post using all previous choices
- Follows structure, applies user profile
- Target: 1,250-3,000 chars, 14+ short paragraphs
- User can request adjustments

**STEP 6: CTA Selection**
- Offers 3 CTAs based on objective
- Types: engagement, save, conversion
- User selects one

**STEP 7: Score & Validation**
- Calculates score (0-10) based on:
  - Hook quality (30%)
  - Structure/length (25%)
  - Data usage (15%)
  - CTA clarity (15%)
  - Algorithm alignment (15%)
- Shows predictive analytics (top 1%, top 5% probability)
- Suggests best day/time to post

**CRITICAL RULE:** If score < 9, system auto-generates improved version. User CANNOT save posts with score < 9.
