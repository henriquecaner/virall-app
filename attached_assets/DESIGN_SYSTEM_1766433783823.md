# Guia de Estilo VIRALL

Este documento contém os tokens de design utilizados no projeto. Para importar como template no Replit, copie o arquivo `tailwind.config.js`.

## 1. Tipografia
**Font Family:** Plus Jakarta Sans
*Import URL:* `https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@300;400;500;600;700;800&display=swap`

## 2. Paleta de Cores

### Brand (Azul LinkedIn / Tech)
Usado para elementos principais, links e identidade da marca.
- **50:** `#eef6fc` (Backgrounds sutis)
- **100:** `#dbeafe`
- **200:** `#bfdbfe`
- **300:** `#93c5fd`
- **400:** `#60a5fa`
- **500:** `#3b82f6` (Botões Hover)
- **600:** `#0A66C2` (**COR PRIMÁRIA** - Azul LinkedIn)
- **700:** `#1d4ed8`
- **800:** `#1e40af`
- **900:** `#1e3a8a`

### Urgency (Laranja)
Usado para escassez, alertas e CTAs de alta conversão.
- **500:** `#FF6B35` (Botões de Ação)
- **600:** `#e85d2e` (Hover)

### Success (Verde)
Usado para provas sociais, checks e mensagens de garantia.
- **500:** `#00C853`
- **600:** `#00a844`

### Dark (Escala de Cinza Premium)
Usado para textos e fundos de alto contraste.
- **900:** `#1E1E1E` (Preto Premium)
- **800:** `#333333`

## 3. Animações Customizadas

### Classes Tailwind
- `animate-pulse-slow`: Pulsação lenta para chamar atenção sem irritar.
- `animate-bounce-subtle`: Pulo suave para setas e indicadores.
- `animate-slide-up`: Entrada suave de baixo para cima (usado no Sticky CTA).
- `animate-[progress-bar-stripes_1s_linear_infinite]`: Efeito de "barbearia" em barras de progresso.

---

**Como usar no Replit:**
1. Crie um novo projeto React.
2. Instale o Tailwind CSS.
3. Substitua o arquivo `tailwind.config.js` pelo arquivo fornecido neste projeto.
4. Adicione a fonte no `index.html` ou `globals.css`.
